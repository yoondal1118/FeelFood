# front
HTML, CSS, JS, Python을 이용한 Flask Web 구축을 위한 Repository 입니다
+ git clone을 한 후에, app.py 파일 실행하면 열립니다
+ 코드 수정을 하시려면, 새로운 branch 생성 후, push 해주시면 다같이 리뷰 후 main에 추가하는 방식으로 하면 좋을 것 같아용
+ 필요한 라이브러리
  + pip install flask
  + pip install pandas
  + pip install python-dotenv
  + pip install google-genai
  + pip install pymysql
  + pip install requests
  + pip install mysql-connector-python
